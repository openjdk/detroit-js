# This is the Detroit javascript engine for Java.

Note: You need to have JDK 25 on the PATH:

```bash
export PATH=/path/to/jdk-25:$PATH"
```

For an interactive shell, please run:

```bash
sh bin/jjs.sh
```

To run .js files:

```bash
sh bin/jjs.sh <script-file>
```

To run java source files that use detroit engine:

```bash
sh bin/run.sh <.java file>
```

The "samples" folder has .js and .java samples.
